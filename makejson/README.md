## Make Json

Write a Go program named makejson.go that performs the following steps:

1. Prompt the user to enter a name.

2. Prompt the user to enter an address.

3. Create a map with the following keys:

```
"name"
"address"
```

4. Store the user-entered values in the map.

5. Use json.Marshal() to convert the map into a JSON object.

6. Print the resulting JSON object.

## Example output:

{"name":"John","address":"123 Main St"}
